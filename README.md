# Day 7 - RESTful API Mini Project

This is a simple RESTful API built using **Flask** as part of IBM Back-End Development Internship Task 7.

## 🔧 Features:
- Retrieve all items (GET)
- Add a new item (POST)
- Update an item (PUT)
- Delete an item (DELETE)
